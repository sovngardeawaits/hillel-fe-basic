Список команд c терминалом

git status - определяет какие файлы в каком состоянии находятся.

git remote -v - показывает общую папку с привязанными репозиториями.

git add - применяет изменения для будущего коммита.

git commit -m - добавление описания изменений в коммит.

git log - показывает все логи над проектом.

git push - создание коммита и отправка на сервер.
