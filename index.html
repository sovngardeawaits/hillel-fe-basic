Селектори та їх типи

  • Назва елементу - будь-який html елемент
  • Класи - .classname (можуть повторюватися)
  • id - #nameid (повинні бути унікальними)
  • Кілька класів - div class="class1 class2" (в одного елемента буде відразу два або більше класів)
  • Вкладеність -

    одночасно встановити стиль для окремого елемента, а також елемента, який знаходиться всередині іншого (наприклад встановити стиль для "p" тільки коли він розташовується всередині контейнера "li")
